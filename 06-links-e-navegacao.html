Links e navegação em html 5

Link para Site Externo

Link externo aponta para um endereço (URL) fora do seu site ou do servidor onde seus arquivos estão hospedados

Como ele funciona?

Para que o navegador saiba que deve sair do seu domínio, o atributo href do elemento <a > deve conter o endereço completo do site externo, incluindo o protocolo (http:// ou https://).

Atributo target="_blank

O atributo target="_blank é usado para abrir o link em uma nova aba ou janela do navegador, permite que o usuário mantenha a pagina atual aberta enquanto visita o site externo.
